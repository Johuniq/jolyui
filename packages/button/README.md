# @jolyui/button

A versatile and accessible button component built with React, Radix UI, and class-variance-authority.

## Installation

```bash
npm install @jolyui/button
```

## Usage

```tsx
import { Button } from "@jolyui/button";

export default function App() {
  return <Button>Click me</Button>;
}
```

## Features

- Multiple variants (default, destructive, outline, secondary, ghost, link)
- Multiple sizes (sm, default, lg, icon)
- Support for polymorphic rendering with `asChild`
- Full TypeScript support
- Accessible by default
